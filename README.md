# 💼 Portfolio

> Un portfolio moderne et responsive présentant mes projets et compétences

[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://portefoliomathisbeugnet.netlify.app/)

## 🌟 Aperçu

Ce portfolio présente mes projets, compétences et expériences de développement. Construit avec Astro à partir d'un template.

## ✨ Fonctionnalités

- **Design Responsive** - Optimisé pour tous les appareils (mobile, tablette, desktop)
- **Navigation Fluide** - Transitions et animations soignées
- **Section Projets** - Présentation détaillée de mes réalisations
- **Formulaire de Contact** - Contactez-moi facilement
- **Mode Sombre/Clair** - Thème adaptatif pour le confort visuel

## 🛠️ Technologies Utilisées

- **Framework**: Astro
- **Frontend**: HTML, CSS, JavaScript
- **Styling**: Tailwind CSS
- **Hébergement**: Netlify

Le site sera accessible à l'adresse `https://portefoliomathisbeugnet.netlify.app/`
